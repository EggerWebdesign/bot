# UDCB

[![CircleCI](https://img.shields.io/circleci/project/github/UDCBOT/bot.svg)](https://circleci.com/gh/UDCBOT)

## Setup

### Requirements
* [NodeJS](https://nodejs.org/en/) (v9.4.0 +)
* Some IDE

### Get started
* Run `npm install`
* Setup `.env` File with your bot token:
    ```
    token = YOUR_TOKEN
    ```
* Setup eslint in your IDE
